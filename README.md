## 📌 About

Cash-Flow is a simple salary and expense tracker built using Vanilla JavaScript (no frameworks).

The goal of this project was to understand how web applications work at a fundamental level — handling user input, updating the UI dynamically, and storing data in the browser so it persists even after refreshing the page.

---

## ✅ Features

### Level 1 — Core

- Set monthly salary
- Add expenses with name and amount
- Display all expenses in a list
- Real-time calculation of remaining balance
- Input validation for empty and invalid values

### Level 2 — Intermediate

- LocalStorage support — data persists after refresh
- Delete functionality — remove any expense instantly
- Chart.js Doughnut Chart — visualizes Expenses vs Remaining Balance

### Bonus (Level 3)

- Budget Alert — triggers when balance drops below 10% of salary
